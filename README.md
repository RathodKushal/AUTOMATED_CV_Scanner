# 📄 Automatic CV Scanner & Summarizer

An AI-powered resume screening system that automatically extracts candidate information, analyzes skills and experience, and generates concise summaries to help recruiters evaluate resumes faster.

## 🚀 Features

- 📂 Upload resumes in PDF format
- 🔍 Extract candidate details automatically
- 💼 Identify skills, education, and work experience
- 🤖 AI-generated resume summaries
- ⚡ Fast and efficient resume screening
- 📊 Structured candidate information output
- 📝 Recruiter-friendly candidate overview
